ให้เป็นระบบติดตามการผลิตออนไลน์

ตั้งแต่ลูกค้าสั่งผลิต → วางแผน → ผลิตแต่ละขั้น → QC → QA → เข้าคลัง ทุกฝ่ายบันทึกและดึงข้อมูลจาก แหล่งเดียวกัน เห็นสถานะงานแบบเรียลไทม์

🚀 ลองเล่น Demo 📊 ดู Dashboard

ปัญหาวันนี้ & สิ่งที่ระบบช่วย

สรุปจากที่คุยกับทีมงานและผังงานที่ได้รับมา

❌ ตอนนี้

  • จดงานลงกระดาษทุกวัน วันละ 20+ รายการต่อฝ่าย
  • ข้อมูลกระจัดกระจาย แต่ละฝ่ายมีของตัวเอง
  • ฐานข้อมูลใช้ Google Sheets ช้าและไม่ปลอดภัย
  • ตามสถานะงานยาก ไม่รู้ว่า job ไหนถึงขั้นไหน
  • คำนวณต้นทุน/KPI ทีหลัง ใช้เวลานาน

✅ หลังมีระบบ

  • บันทึกผ่านคอม/มือถือ เลือกจากรายการ ไม่ต้องเขียนซ้ำ
  • ข้อมูลกลางชุดเดียว ทุกฝ่ายเห็นตรงกันแบบเรียลไทม์
  • ฐานข้อมูลบน Cloud ปลอดภัย รองรับผู้ใช้พร้อมกัน
  • เห็นสถานะทุก job บนบอร์ดเดียว (Pending Order)
  • Daily Report & ต้นทุน/KPI สรุปอัตโนมัติ

Flow การผลิตที่ระบบครอบคลุม

อิงผังงานจริงจากเอกสารที่ได้รับ

เตรียมยา→ ผสม→ ตอก / แคปซูล→ บรรจุ (Blister/Strip/ซอง/ขวด)→ QC→ QA→ FG (คลังสำเร็จรูป)

โมดูลหลักในระบบ

Dashboard / KPI

ภาพรวมผู้บริหาร: จำนวนงานตามสถานะ, % on-plan, ต้นทุนแรงงาน (DL)

แผนการพัฒนา (Roadmap)

แบ่งเป็นช่วง ทำทีละก้อน เห็นผลทุกขั้น

ช่วงที่ 1 · Demo
HTML prototype (หน้านี้) เพื่อเก็บ feedback จากผู้บริหาร — ทำแล้ว
ช่วงที่ 2 · วางโครง
ตั้งโปรเจค Next.js + เชื่อม Supabase + ขึ้น Vercel
ช่วงที่ 3 · ฐานข้อมูล
สร้างตาราง + ข้อมูลหลัก (ยา, เครื่องจักร)
ช่วงที่ 4 · ผู้ใช้
ระบบ login + สิทธิ์ตามฝ่าย (ผลิต/QC/QA/คลัง/บัญชี)
ช่วงที่ 5 · Board + Job
สร้าง/ติดตาม Pending Order ใช้งานจริง
ช่วงที่ 6 · บันทึกผลิต
บันทึก in-out แต่ละ station ลงฐานข้อมูล
ช่วงที่ 7 · รายงาน
Daily Report + Dashboard/KPI + ต้นทุน
ช่วงที่ 8 · ใช้งานจริง
ทดสอบกับข้อมูลจริง + ปรับ + เปิดใช้

ค่าใช้จ่าย & ความเสี่ยง

จุดเด่น: เริ่มต้นไม่มีค่า license รายปีแบบซื้อโปรแกรมสำเร็จรูป

รายการช่วง Demoใช้งานจริง (ระยะแรก)
Hosting เว็บ (Vercel)ฟรีฟรี (free tier)
ฐานข้อมูล (Supabase)ไม่ต้องใช้ฟรี จนกว่าข้อมูล/ผู้ใช้จะมาก
ค่า license รายปี❌ ไม่มี❌ ไม่มี (เป็นระบบของเราเอง)
ค่าดูแล/พัฒนาเพิ่ม—ปรับแก้เองได้ ไม่ต้องผูกกับเจ้าเดียว

💡 เมื่อผู้ใช้/ข้อมูลมากขึ้น อาจเสียค่า Cloud รายเดือนตามจริง (หลักร้อย–พันบาท/เดือน) ซึ่งยังถูกกว่าและยืดหยุ่นกว่าการซื้อโปรแกรมสำเร็จรูปที่ต้องจ่ายรายปีและแก้เองไม่ได้
